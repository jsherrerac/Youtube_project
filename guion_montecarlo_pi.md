# Guion — "El número que dibuja el azar"
### DotCorzo · Segundo video largo · Tema: Método de Montecarlo (estimar π) · Objetivo: 5:00–5:45 min

---

## Antes de grabar — léelo una vez

**Ritmo:** habla más lento de lo que crees. Las pausas `//` son reales — respira ahí. El clímax (Beat 5) se gana con **silencio**: deja que los puntos llenen el cuadrado y que el número se asiente solo, sin voz encima.

**Voz:** tranquila, curiosa, como mostrándole algo increíble a un amigo. No dictes clase. Graba por beats y empalmas después.

**La regla de oro — consistencia de color** (amarra el video y tu banco de animaciones):
- **Punto DENTRO del círculo (acierto) → CIAN** — el color del "sí".
- **Punto FUERA (fallo) → CORAL/ROJO** — el color del "no".
- **Borde (cuadrado + cuarto de círculo) → BLANCO.**
- **El número estimado de π y la línea de π real → AMARILLO/DORADO.**
- Fondo **negro**. Colores idénticos del minuto 1 al final. (Misma familia de paleta que el video del seno: cian + coral + amarillo + blanco.)

**Duración:** narración corta a propósito (~700 palabras). La lluvia de puntos y la convergencia cargan el tiempo. NO la rellenes con voz — el video respira por los ojos.

## BEAT 1 — El gancho ⏱ 0:00–0:25

**🎙️ Narración:**
> Esto es azar puro. //
> Puntos cayendo sin ningún orden, // sin patrón, // sin reglas. //
> Y sin embargo, // de este desorden total //
> va a salir uno de los números más precisos de las matemáticas. //
> Pi. // Tres punto uno cuatro uno cinco nueve... //
> No lo vamos a calcular con una fórmula. //
> Lo vamos a encontrar // tirando puntos al azar. //
> ¿Cómo es posible que el azar // dibuje un número exacto? //
> Vamos a verlo.

**🎬 En pantalla:**
- Abre con puntos cayendo al azar sobre negro, sin estructura, sin orden.
- Al nombrar "pi": insinúa el número grande y borroso de fondo, o un destello.
- Antes del corte: aparece tenue el cuadrado con el cuarto de círculo (preview de lo que viene). Corte de tensión.

---

## BEAT 2 — El tablero ⏱ 0:25–1:30

**🎙️ Narración:**
> Empecemos con lo más simple. //
> Un cuadrado. // De lado uno. //
> Y dentro, // un cuarto de círculo // de radio uno, // pegado a esta esquina. //
> Ahora, // tiramos un punto al azar dentro del cuadrado. //
> En cualquier lado. // Totalmente aleatorio. //
> Y nos hacemos una sola pregunta: //
> ¿cayó dentro del círculo... // o fuera? //
> ¿Cómo lo sabemos? // Medimos su distancia a la esquina. //
> Si esa distancia es menor que el radio, // está dentro. // Lo pintamos de cian. //
> Si es mayor, // está fuera. // Lo pintamos de coral. //
> Un punto. // Una pregunta. // Dentro o fuera. //
> Tiremos unos cuantos más.

**🎬 En pantalla:**
- `Create` del cuadrado blanco (lado 1). Luego `Create` del cuarto de círculo blanco (radio 1) desde la esquina.
- Suelta UN punto. Dibuja el segmento de su distancia a la esquina (la línea radial) y compáralo con el radio → si entra, el punto y el segmento se vuelven **cian**; si no, **coral**.
- Repite con ~8–12 puntos, uno a uno, cada uno coloreándose según dentro/fuera. Velocidad creciente al final.

---

## BEAT 3 — La razón escondida ⏱ 1:30–2:40

**🎙️ Narración:**
> Y acá está la idea que lo cambia todo. //
> Si los puntos caen de verdad al azar, // repartidos parejo por todo el cuadrado, //
> entonces la fracción que cae dentro del círculo //
> no es suerte. //
> Es, exactamente, // la proporción entre las dos áreas. //
> El área del cuarto de círculo // dividida // entre el área del cuadrado. //
> El cuadrado de lado uno // tiene área uno. //
> Y el cuarto de círculo de radio uno // tiene área // pi // sobre cuatro. //
> Entonces la fracción de puntos que caen dentro //
> se acerca a // pi // sobre cuatro. //
> Y si despejamos... //
> pi // es, simplemente, // cuatro // por los puntos de adentro // sobre el total. //
> Ya no necesitamos saber qué es pi. //
> Solo necesitamos // contar puntos.

**🎬 En pantalla:**
- Sombrea el área del cuarto de círculo (cian translúcido) y la del cuadrado (contorno blanco).
- Escribe la razón: `área del cuarto de círculo / área del cuadrado` = `(π/4) / 1`.
- Construye la fórmula final en pantalla, heredando colores: `π ≈ 4 · (dentro / total)`. "dentro" en cian, el resultado en amarillo.
- Muestra la fórmula evaluándose con los conteos actuales del Beat 2 (un valor todavía impreciso).

---

## BEAT 4 — La ley de los grandes números ⏱ 2:40–3:50

**🎙️ Narración:**
> Pero hay una trampa. //
> Con pocos puntos, // el azar es caprichoso. //
> Mira: // con diez puntos, // la cuenta da cualquier cosa. // Lejos de pi. //
> Con cien, // se va acercando. //
> Con mil, // ya casi le apunta. //
> Esto tiene nombre: // la ley de los grandes números. //
> Un puñado de intentos // miente. //
> Pero miles de intentos // dicen la verdad. //
> Mientras más puntos tiramos, //
> más se estabiliza la respuesta. //
> Sigamos tirando... // y miremos el número.

**🎬 En pantalla:**
- Tira puntos en tandas: 10 → la estimación salta lejos de π. 100 → se acerca. 1000 → casi le apunta. El contador de π se actualiza en vivo (amarillo).
- A un lado, aparece una gráfica de convergencia: eje x = número de puntos (N), eje y = estimación; una línea que tiembla y se va apretando hacia una **línea punteada amarilla en π**.

---

## BEAT 5 — La convergencia · CLÍMAX ⏱ 3:50–5:00

**🎙️ Narración:**
> Ahora soltamos miles. //
> Decenas de miles. //
> Y miramos solo // ese número, // subiendo, // ajustándose... //
> tres punto cero... // tres punto uno... // tres punto uno cuatro... //
> //
> *(silencio — deja que los puntos llenen el cuadrado y el número se asiente)* //
> //
> Ahí está. //
> Pi. //
> No lo calculamos nosotros. //
> No usamos ninguna fórmula de pi. //
> Lo dibujó // el azar. //
> Miles de puntos sin ningún orden, // juntos, // revelan // un número perfecto. //
> Eso // es el método de Montecarlo.

**🎬 En pantalla:**
- Lluvia rápida de miles de puntos; el cuadrado se llena de cian/coral formando nítidamente el cuarto de círculo.
- El número de π (grande, dorado) hace tic hacia `3.14159…`. La línea de convergencia besa la línea punteada de π.
- ESTE es el momento: cuando el número se asienta, **2–3 segundos sin texto ni voz**. Solo el cuadrado lleno y el número clavado.

---

## BEAT 6 — El pago y el puente ⏱ 5:00–5:45

**🎙️ Narración:**
> Y esto no se queda en pi. //
> El truco es siempre el mismo: //
> cuando algo es demasiado difícil de calcular con una fórmula, //
> lo bombardeas con azar // y cuentas qué pasa. //
> Así se estima // el área de formas imposibles, //
> el precio de cosas en finanzas, //
> el comportamiento de partículas en física, //
> el riesgo de una decisión... //
> hasta cosas dentro de la inteligencia artificial. //
> Azar // más // conteo. //
> //
> Y cada vez que ves una bolita rebotando al azar en mis simulaciones, //
> por dentro // está corriendo // un pequeño Montecarlo. //
> Desorden afuera... // orden adentro. //
> Ahora // ya sabes verlo.

**🎬 En pantalla:**
- Recap en 3 imágenes encadenadas: puntos al azar → la razón de áreas → el número π.
- Imagen concreta de "otras aplicaciones": una forma irregular rara siendo medida por puntos al azar (área estimada por conteo).
- En "y de esto": hueco/placeholder de ~5s (rectángulo negro con comentario en el código) donde insertas en edición un clip de una simulación de Pygame.
- Cierre: logo/handle DotCorzo + "donde la física se vuelve visible".
