# Guion — "La onda escondida en un triángulo"
### DotCorzo · Primer video largo · Objetivo: 6:30–7:00 min

---

## Antes de grabar — léelo una vez

**Ritmo:** habla más lento de lo que crees que debes. Las pausas marcadas con `//` son reales — respira ahí. El clímax (Beat 5) se gana con silencio: deja que la animación trace la onda sin hablar encima.

**Voz:** tono tranquilo, curioso, como si le estuvieras mostrando algo a un amigo, no dictando clase. Graba por beats (uno a la vez) y empalmas después.

**La regla de oro — consistencia de color** (esto amarra todo el video y tu futuro banco de animaciones):
- **Cateto opuesto / altura → AZUL** → por eso el seno es azul → por eso la onda es azul.
- **Cateto adyacente / ancho → VERDE** → por eso el coseno es verde.
- **Hipotenusa / radio → BLANCO.**
- Fondo **negro**. Mantén estos colores idénticos del minuto 1 al final. El espectador nunca debe preguntarse qué representa qué.

**Duración:** la narración es corta a propósito (~750 palabras). Los giros, los trazados y las pausas cargan el resto del tiempo. No la rellenes — el video respira por los ojos, no por la boca.

## BEAT 1 — El gancho ⏱ 0:00–0:25

**🎙️ Narración:**
> Esto es una onda. //
> Describe el sonido que escuchas, // la luz que ves, // un resorte que vibra... //
> y hasta una pelota rebotando. //
> Está en todas partes. //
> Pero lo más extraño no es *dónde* aparece. //
> Es *de dónde sale*. //
> Porque toda esta curva — tan suave, tan complicada — //
> nace de la figura más simple que existe: //
> un triángulo. //
> ¿Cómo es posible? //
> Vamos a verlo.

**🎬 En pantalla:**
- Abre con una onda senoidal azul moviéndose suave, sola, sobre fondo negro.
- Mientras nombras sonido/luz/r  esorte/pelota: íconos mínimos apareciendo y desvaneciéndose (no satures — uno a la vez).
- En "un triángulo": la onda se encoge/transforma hacia un triángulo pequeño. Corte de tensión.

---

## BEAT 2 — El triángulo rectángulo ⏱ 0:25–1:35

**🎙️ Narración:**
> Empecemos por el triángulo más útil de todos: //
> el triángulo rectángulo. //
> Tiene un ángulo de noventa grados, //
> y con solo dos de ellos puedes construir cualquier otro triángulo. //
> Así que si entiendes este, // los entiendes todos. //
> Fijémonos en uno de sus ángulos. // Lo llamaremos *alfa*. //
> El lado más largo, frente al ángulo recto, es la **hipotenusa**. //
> El lado de enfrente de alfa, el **cateto opuesto**. //
> Y el que toca el ángulo, el **cateto adyacente**. //
> Ahora, lo interesante. //
> Si tomas dos de estos lados // y divides su longitud, //
> obtienes un número. //
> Y ese número // no cambia. //
> Puedes hacer el triángulo enorme // o diminuto — //
> mientras el ángulo alfa sea el mismo, //
> la proporción es exactamente la misma. //
> Lo único que puede cambiar ese número... //
> es el ángulo.

**🎬 En pantalla:**
- Dibuja (Create) un triángulo rectángulo blanco. Marca el ángulo recto con su cuadradito.
- Resalta el ángulo `α` con un arco.
- Etiqueta los tres lados **con sus colores**: hipotenusa blanca, opuesto **azul**, adyacente **verde**. Cada label aparece cuando lo nombras.
- En "no cambia": muestra DOS triángulos, uno grande y uno chico, mismo α. Una proporción (ej. opuesto/hipotenusa) escrita al lado da el mismo número en ambos. Resáltalo.
- En "es el ángulo": haz crecer/encoger α y muestra el número cambiando en vivo.

---

## BEAT 3 — De proporción a función ⏱ 1:35–2:50

**🎙️ Narración:**
> Y ahí está la idea clave. //
> Si esa proporción solo cambia cuando cambia el ángulo, //
> entonces podemos verla como una **función** del ángulo. //
> Le das un ángulo... // y te devuelve un número. //
> Estas funciones ya tienen nombre. //
> La proporción entre el cateto opuesto y la hipotenusa //
> se llama **seno**. //
> Y la del adyacente con la hipotenusa, //
> **coseno**. //
> Hagamos un truco para simplificarlo todo. //
> Fijemos la hipotenusa en uno. //
> Porque si la hipotenusa vale uno... //
> entonces el seno de alfa //
> es, simplemente, // la longitud del cateto opuesto. //
> Sin divisiones. //
> El seno deja de ser una cuenta // y se vuelve // una altura.

**🎬 En pantalla:**
- Muestra la proporción `opuesto / hipotenusa` y, debajo, la flecha "ángulo → número" (idea de función).
- Escribe `sin(α)` en **azul** (combina con el opuesto) y `cos(α)` en **verde**. La fórmula hereda el color del lado.
- En "fijemos la hipotenusa en uno": la hipotenusa se ajusta a longitud 1, aparece el "1" sobre ella.
- En "es una altura": colapsa visualmente `sin(α) = opuesto`, resaltando el cateto opuesto azul como una barra de altura.

---

## BEAT 4 — El círculo unitario ⏱ 2:50–4:10

**🎙️ Narración:**
> Y si la hipotenusa siempre vale uno... //
> ¿qué pasa si la dejamos girar? //
> El extremo de esa hipotenusa //
> traza un círculo. //
> Un círculo de radio uno — //
> lo llamamos el **círculo unitario**. //
> Y mira lo que tenemos ahora. //
> El radio // es la hipotenusa. //
> La altura del punto sobre el círculo // es el cateto opuesto. //
> O sea... // el seno del ángulo. //
> Y el ancho // es el cateto adyacente — //
> el coseno. //
> Cada punto de este círculo //
> esconde un triángulo. //
> Giremos el ángulo despacio... //
> y miremos solo la altura. //
> Sube... // llega arriba... // baja... // cruza el centro... // y vuelve.

**🎬 En pantalla:**
- La hipotenusa blanca empieza a girar desde su vértice; su punta deja un rastro que forma el círculo unitario.
- Dentro del círculo, dibuja el triángulo inscrito: radio (blanco), altura (**azul**, vertical), ancho (**verde**, horizontal). Se actualizan al girar.
- En "esconde un triángulo": gira a un par de ángulos distintos para que se vea el triángulo cambiando.
- En "miremos solo la altura": atenúa todo menos la barra azul de altura. Gira lento una vuelta; la barra azul sube y baja.

## BEAT 5 — El desenrollo · CLÍMAX ⏱ 4:10–5:40

**🎙️ Narración:**
> Vamos a hacer una sola cosa más. //
> A la derecha, // dibujemos unos ejes. //
> En el eje horizontal, // el ángulo. //
> En el vertical, // la altura del punto. //
> Ahora giramos el punto... //
> y cada altura // la marcamos en su ángulo. //
> //
> *(silencio — deja que la animación trace la onda completa)* //
> //
> Ahí está. //
> La onda. //
> No la dibujamos nosotros. //
> La dibujó el círculo. //
> Eso // es el seno. //
> Y el coseno es la misma curva — //
> solo que mide el ancho en vez de la altura, //
> así que va corrida noventa grados. //
> La misma onda... // un paso atrás.

**🎬 En pantalla:**
- Aparecen los ejes a la derecha: x etiquetado con `π`, `2π`; y de −1 a 1.
- El punto gira en el círculo (izquierda); una **línea punteada** conecta su altura con el punto correspondiente en los ejes.
- Conforme α avanza de 0 a 2π, se **traza la senoidal azul sola**. Este es EL momento — sin narración, solo el trazo. Déjalo respirar 2–3 segundos.
- Etiqueta la curva `sin(α)` en azul.
- Luego traza el coseno **verde** en el mismo eje, corrido π/2. Resalta el desfase con una flechita.

## BEAT 6 — El pago y el puente ⏱ 5:40–6:45

**🎙️ Narración:**
> Y eso es todo. //
> Una onda... // salida de un triángulo... // girando en un círculo. //
> Pero esta curva no se queda en el papel. //
> Si estiras su altura, //
> cambias el volumen de un sonido. //
> Si la haces girar más rápido, //
> cambias su tono. //
> Es la misma matemática // detrás de la música, // de la luz, //
> de un resorte que vibra... //
> y de esto. //
> //
> Cada vez que una pelota sube y baja en mis simulaciones, //
> hay un círculo escondido, // dibujando una onda. //
> Ahora // ya sabes verla.

**🎬 En pantalla:**
- Recapitulación rápida en 3 imágenes encadenadas: triángulo → círculo → onda.
- Muestra la **fórmula viva** `A·sin(k·α + φ)`: al subir `A`, la onda se estira (amplitud); al subir `k`, se comprime (frecuencia). Cámbialos en vivo.
- En "y de esto": corta a un clip de una de tus simulaciones de bolita rebotando (Pygame). Superpón sutilmente la senoidal sobre el movimiento vertical de la bola para mostrar el círculo/onda escondido.
- Cierre: logo/handle DotCorzo + "donde la física se vuelve visible".
